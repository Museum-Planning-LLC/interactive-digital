# Template matrix — four Digital Exhibits archetypes

**Museum Planning LLC** · internal · 90-day repository flush

These four rows are **delivery templates**, not products. Not sold cold. Not shown publicly during the flush period. Each proves a distinct **participation model** under the same Digital Exhibits category (kiosk · full-wall · full-room).

Compare: [form-factors.md](./form-factors.md) · [principles.md](./principles.md)

---

## Matrix

| # | Catalog entry | Domain | Default form factor | Participation | Visitor does | Institution supplies |
|---|---------------|--------|---------------------|---------------|--------------|----------------------|
| **1** | [shared-ground](../catalog/shared-ground.md) | Humanities / political memory | Full-room | `social-geometry` | Moves with others; relationship surfaces archive | Plural photo/archive threads, place |
| **2** | [flow-field-grid](../catalog/flow-field-grid.md) | Science | Full-wall (or kiosk) | `simulation` | Adjusts parameters; reads field behavior | Phenomenon framing (optional); sim tuning |
| **3** | [local-history-place](../catalog/local-history-place.md) | Local history | Full-wall or kiosk | `presence` + place | Moves through map/layers; place reveals material | Geo-tagged archive, streets, eras |
| **4** | [humanities-field](../catalog/humanities-field.md) | Humanities (non-convergence) | Kiosk or full-wall | `mixed` (compare) | Pairs / juxtaposes; no single verdict | Objects, texts, voices as nodes |

**Rule:** Template #4 must **not** duplicate Shared Ground’s convergence grammar. Different hook, same repo.

---

## Layer checklist (all templates)

Use the same honest status in every catalog entry:

| Layer | Job |
|-------|-----|
| **Core** | Behavior engine (sim, archive logic, map, graph) |
| **Participation** | What visitor controls |
| **Navigation** | Attract → engage → reset; staff mode if needed |
| **Presentation** | Type, palette, layout, sound |
| **Install shell** | Fullscreen, offline, kiosk durability, health ping |

**90-day goal:** Core + catalog + example JSON for each row. Navigation / presentation / install shell documented as gaps until a menu item funds work.
